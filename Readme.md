JoZHD
======

JoZHD is a texture pack for minetest. It is mostly at 1024x1024px.

**This is the 'Screens' branch which is used to store screenshots.**
--------------------------------------------------------------------
The textures are at https://github.com/UgnilJoZ/jozhd.
